# 数据分析报告模板

> **使用说明**：本模板适用于数据洞察、语义分析、数据挖掘等各类分析任务，可根据实际需求灵活调整章节。数据挖掘任务需保留「五、模型构建与评估」章节；描述性分析任务可省略该章节。

---

# 执行摘要

## 分析背景与目标
- **分析目的**：简要说明本次分析要解决的核心问题或达成的目标
- **业务背景**：描述分析的业务场景和相关背景信息
- **分析范围**：明确分析涉及的数据范围、时间范围、业务领域等

## 核心发现（Key Findings）
> 用3-5个要点概括最重要的发现，每个发现应包含数据支撑，每个点不小于300字

1. **发现一**：简要描述，如"A指标较上期增长X%，主要受B因素驱动"
2. **发现二**：简要描述，如"C群体与D群体在E指标上存在显著差异（p<0.05）"
3. **发现三**：简要描述，如"F因素与G结果呈现强正相关（r=0.85）"

## 主要建议
> 基于分析发现提出的2-3条可操作建议

1. **建议一**：具体行动建议及预期效果
2. **建议二**：具体行动建议及预期效果

# 二、分析方法

## 2.1 分析框架
```
问题定义 → 数据预处理 → 探索性分析 → 模型构建（挖掘任务）→ 结果解释 → 业务建议
```

## 2.2 统计方法说明

### 描述性统计
| 统计量 | 计算方法 | 适用场景 |
|--------|----------|----------|
| 集中趋势 | 均值、中位数、众数 | 数值型变量分布中心 |
| 离散程度 | 标准差、四分位距、变异系数 | 数值型变量离散程度 |
| 频数统计 | 计数、百分比 | 分类型变量分布 |

### 推断性统计
| 分析方法 | 适用条件 | 应用场景 |
|----------|----------|----------|
| Pearson相关 | 连续变量，线性关系 | 变量间线性相关程度 |
| Spearman相关 | 有序变量或非线性关系 | 变量间单调相关程度 |
| 独立样本t检验 | 两组比较，正态分布 | 两组均值差异检验 |
| 单因素方差分析 | 多组比较，正态分布 | 多组均值差异检验 |
| 卡方检验 | 分类变量 | 分类变量独立性检验 |

## 2.3 分析工具
- **数据处理**：Python (pandas, numpy)
- **统计分析**：Python (scipy, statsmodels, scikit-learn)
- **可视化**：Python (matplotlib, seaborn, plotly)

> 描述性分析任务可省略本章节。

## 5.1 方法选择

| 分析阶段 | 方法名称 | 方法描述 | 应用场景 | 参数设置 |
|----------|----------|----------|----------|----------|
| 探索性分析 | 描述性统计 | 计算基本统计量 | 了解数据分布 | - |
| 建模阶段 | 随机森林 | 基于决策树的集成学习 | 分类/回归 | n_estimators=100 |
| 建模阶段 | K-means | 基于距离的聚类 | 客户分群 | n_clusters=3 |

## 5.2 模型评估

| 评估指标 | 训练集 | 验证集 | 测试集 | 说明 |
